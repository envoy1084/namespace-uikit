---
title: Getting started
description: Install Namespace UIKit and its global styles.
---

Install the package with your package manager:

<Tabs items={["pnpm", "npm", "yarn"]} groupId="package-manager" persist>
  <Tab value="pnpm">`pnpm add @thenamespace/uikit`</Tab>
  <Tab value="npm">`npm install @thenamespace/uikit`</Tab>
  <Tab value="yarn">`yarn add @thenamespace/uikit`</Tab>
</Tabs>

Import the stylesheet once in the root of your application:

```tsx
import "@thenamespace/uikit/styles";
```

Then import components from the package root or a one-level component entry point:

```tsx
import { Button } from "@thenamespace/uikit/button";

export function SaveButton() {
  return <Button>Save changes</Button>;
}
```
