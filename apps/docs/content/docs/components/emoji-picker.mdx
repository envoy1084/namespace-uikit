## Anatomy

Import the EmojiPicker component and access all parts using dot notation.

```tsx
import { EmojiPicker } from "@thenamespace/uikit";

<EmojiPicker>
  <EmojiPicker.Trigger>
    <EmojiPicker.Value />
  </EmojiPicker.Trigger>
  <EmojiPicker.Popover>
    <EmojiPicker.Content>
      <EmojiPicker.SkinTonePicker>
        <EmojiPicker.SkinToneTrigger />
        <EmojiPicker.SkinToneContent>
          <EmojiPicker.SkinToneOption />
        </EmojiPicker.SkinToneContent>
      </EmojiPicker.SkinTonePicker>
      <EmojiPicker.Grid>
        <EmojiPicker.Item />
      </EmojiPicker.Grid>
      <EmojiPicker.Footer />
    </EmojiPicker.Content>
  </EmojiPicker.Popover>
</EmojiPicker>;
```

## CSS Classes

### Base & Size Classes

- `.emoji-picker__popover` ‚Äî Floating panel containing the picker. Has `bg-overlay`, `shadow-overlay`, and `rounded-2xl` with enter/exit animations.
- `.emoji-picker__popover--sm` ‚Äî Small popover (`240px √ó 280px`).
- `.emoji-picker__popover--md` ‚Äî Medium popover (`280px √ó 350px`). Default.
- `.emoji-picker__popover--lg` ‚Äî Large popover (`320px √ó 420px`).
- `.emoji-picker__item--sm` ‚Äî Small emoji item (`font-size: 20px`).
- `.emoji-picker__item--md` ‚Äî Medium emoji item (`font-size: 24px`). Default.
- `.emoji-picker__item--lg` ‚Äî Large emoji item (`font-size: 28px`).

### Element Classes

- `.emoji-picker__trigger` ‚Äî Unstyled trigger button with press-scale animation and interactive cursor.
- `.emoji-picker__value` ‚Äî Displays the currently selected emoji inside the trigger.
- `.emoji-picker__content` ‚Äî Flex wrapper inside the popover holding the search input and grid.
- `.emoji-picker__grid` ‚Äî Virtualized scrollable emoji grid with `overflow-auto` and `scroll-padding`.
- `.emoji-picker__item` ‚Äî Individual emoji cell in the grid. Centered flex with rounded hover/focus states.
- `.emoji-picker__footer` ‚Äî Absolutely positioned bottom bar with `bg-surface`, `border-top`, and muted text.
- `.emoji-picker__empty` ‚Äî Empty state centered text shown when no emojis match the search.
- `.emoji-picker__skin-tone-picker` ‚Äî Trigger button for the skin tone selector (`size-6`, rounded).
- `.emoji-picker__skin-tone-options` ‚Äî Row of skin tone swatches inside a popover dialog.
- `.emoji-picker__skin-tone-option` ‚Äî Individual skin tone swatch button with selection highlight.

### Interactive States

- **Entering**: `[data-entering="true"]` on `.emoji-picker__popover` ‚Äî `fade-in` + `zoom-in-95` (150ms).
- **Exiting**: `[data-exiting="true"]` on `.emoji-picker__popover` ‚Äî `fade-out` + `zoom-out-95` (100ms).
- **Trigger pressed**: `[data-pressed="true"]` on `.emoji-picker__trigger` ‚Äî `scale(0.97)`.
- **Trigger focus visible**: `[data-focus-visible="true"]` on `.emoji-picker__trigger` ‚Äî applies focus ring.
- **Trigger disabled**: `[aria-disabled="true"]` on `.emoji-picker__trigger` ‚Äî applies disabled styles.
- **Item hover**: `[data-hovered="true"]` on `.emoji-picker__item` ‚Äî applies `bg-default`.
- **Item focused**: `[data-focused="true"]` on `.emoji-picker__item` ‚Äî applies `bg-default`.
- **Item pressed**: `[data-pressed="true"]` on `.emoji-picker__item` ‚Äî applies `bg-default-hover`.
- **Item selected**: `[data-selected="true"]` on `.emoji-picker__item` ‚Äî applies `bg-default-hover`.
- **Item focus visible**: `[data-focus-visible="true"]` on `.emoji-picker__item` ‚Äî applies focus ring.
- **Skin tone selected**: `[data-selected="true"]` on `.emoji-picker__skin-tone-option` ‚Äî applies `bg-default-hover`.
- **Reduced motion**: `motion-reduce:animate-none` and `motion-reduce:transition-none` on animated elements.

## API Reference

### EmojiPicker

The root provider. Wraps a RAC `Select` internally.

| Prop   | Type                   | Default | Description                                                      |
| ------ | ---------------------- | ------- | ---------------------------------------------------------------- |
| `size` | `"sm" \| "md" \| "lg"` | `"md"`  | Size variant controlling trigger, popover, and emoji dimensions. |

Also supports all RAC [Select](https://react-spectrum.adobe.com/react-aria/Select.html) props.

### EmojiPicker.Trigger

Button that opens the emoji popover.

Also supports all RAC [Button](https://react-spectrum.adobe.com/react-aria/Button.html) props.

### EmojiPicker.Value

Displays the currently selected emoji inside the trigger.

Also supports all RAC [SelectValue](https://react-spectrum.adobe.com/react-aria/Select.html#selectvalue) props.

### EmojiPicker.Popover

The floating panel containing the picker content.

| Prop        | Type        | Default    | Description                                  |
| ----------- | ----------- | ---------- | -------------------------------------------- |
| `offset`    | `number`    | `4`        | Distance from the trigger in pixels.         |
| `placement` | `Placement` | `"bottom"` | Preferred placement relative to the trigger. |

Also supports all RAC [Popover](https://react-spectrum.adobe.com/react-aria/Popover.html) props.

### EmojiPicker.Content

Flex wrapper inside the popover. Wraps an internal `Autocomplete` for search filtering.

| Prop     | Type                                                 | Default                   | Description                              |
| -------- | ---------------------------------------------------- | ------------------------- | ---------------------------------------- |
| `filter` | `(textValue: string, inputValue: string) => boolean` | Case-insensitive contains | Custom filter function for emoji search. |

### EmojiPicker.Grid

Virtualized scrollable emoji grid using RAC `Virtualizer` with `GridLayout`.

| Prop               | Type                                                              | Default                      | Description                                                  |
| ------------------ | ----------------------------------------------------------------- | ---------------------------- | ------------------------------------------------------------ |
| `layoutOptions`    | `{ minItemSize?, maxItemSize?, minSpace?, preserveAspectRatio? }` | `36√ó36` cells, `2√ó2` spacing | Virtualizer grid layout options for item sizing and spacing. |
| `renderEmptyState` | `() => ReactNode`                                                 | ‚Äî                            | Custom empty state when no emojis match.                     |

Also supports all RAC [ListBox](https://react-spectrum.adobe.com/react-aria/ListBox.html) props (except `layout`).

### EmojiPicker.Item

An individual emoji cell in the grid.

Also supports all RAC [ListBoxItem](https://react-spectrum.adobe.com/react-aria/ListBox.html#listboxitem) props.

### EmojiPicker.Footer

Content area below the grid (e.g., selected emoji preview or skin tone picker). Renders a `<div>`.

### EmojiPicker.SkinTonePicker

Root provider for the skin tone selector. Manages a `Popover` internally.

| Prop           | Type                      | Default     | Description                          |
| -------------- | ------------------------- | ----------- | ------------------------------------ |
| `value`        | `string`                  | ‚Äî           | Controlled skin tone value.          |
| `defaultValue` | `string`                  | `"default"` | Default skin tone (uncontrolled).    |
| `onChange`     | `(value: string) => void` | ‚Äî           | Callback when the skin tone changes. |

### EmojiPicker.SkinToneTrigger

Button that opens the skin tone popover. Shows the current skin tone emoji.

| Prop         | Type                  | Default              | Description                                             |
| ------------ | --------------------- | -------------------- | ------------------------------------------------------- |
| `tones`      | `EmojiSkinToneItem[]` | `EMOJI_SKIN_TONES`   | Skin tone data for resolving the current value's emoji. |
| `aria-label` | `string`              | `"Select skin tone"` | Accessible label.                                       |

### EmojiPicker.SkinToneContent

Popover content for the skin tone selector.

| Prop        | Type        | Default        | Description                |
| ----------- | ----------- | -------------- | -------------------------- |
| `offset`    | `number`    | `4`            | Distance from the trigger. |
| `placement` | `Placement` | `"bottom end"` | Popover placement.         |

### EmojiPicker.SkinToneOption

An individual skin tone swatch button.

| Prop | Type     | Default | Description                                                              |
| ---- | -------- | ------- | ------------------------------------------------------------------------ |
| `id` | `string` | ‚Äî       | Skin tone identifier (e.g., `"default"`, `"light"`, `"dark"`). Required. |
